Перед использованием данной программы убедитесь, что на Вашем компьютере установлен JDK версии 11 или более поздней версии.
Чтобы проверить версию JDK, запустите командную строку, введите строку "java -version" и нажмите enter.

Как пользоваться программой pcoder?

1) Запишите настройки в файл sig_settings.json. В данном файле соблюдайте формат json. Все числа необходимо записывать в виде строки, в кавычках. Параметры linetInt и linetFloat должны быть равны. Пример:
{
  "linea" : [
    "0.0",
    "2.5",
    "0.0"
  ],
  "linetInt" : [
    "5",
    "5",
    "5"
  ],
  "linetFloat" : [
    "5.0",
    "5.0",
    "5.0"
  ],
  "linenmb" : "3",
  "repeatcycle" : "3"
}
2) В адресной строке проводника, где открыта папка с данной программой введите cmd и нажмите enter. В запустившейся командной строке введите строку "java -jar pcoder.jar" и нажмите enter;
3) После выполнения данной программы параметры в битовом представлении появятся в файле sim_params.txt, который будет прочитан программой тестового окружения в Vivado.
